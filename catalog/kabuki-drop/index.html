Kabuki Drop

Stunning Kabuki Drop is a great way to add a suggestion of color to any interior without overwhelming impact. This pattern reads like a smaller mosaic pattern and is a great companion piece to some of the bolder larger patterns currently trending. Modern, Asian, Contemporary, Industrial, Nordic, this pattern really does it all.

Suitable for interior wall, countertop and low-traffic floors. Exterior use in non-freezing climates only. Tile dimensions are 6" x 6" and 8" x 8".
